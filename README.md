A ball game I developed in high school, playable only on macOS. The source code has been lost, but the application is still available :D

The game includes multiple difficulty levels:

|        Levels        |
| :-------------------: |
|     Level 1: Easy     |
|    Level 2: Medium    |
|     Level 3: Hard     |
| Level 4: Easy (foggy) |
| Level 5: Hard (foggy) |

During gameplay, the player must avoid black obstacles, colliding with them will restart the current level.
